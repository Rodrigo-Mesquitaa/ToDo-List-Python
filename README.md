# ToDo-List-Python
Projeto ToDo em Python gerando tarefas simples.

Adicionando Itens
Use todo.py add <text>.

Itens de listagem
Usar todo.py list

Removendo itens acabados
Use todo.py done <id> Para obter o ID, usetodo.py list

Uso com várias listas
O nome de uma lista deve ser uma palavra.

Adicionando itens a uma lista específica
Usar todo.py addto <listname> <text>

Listando itens de uma certa lista
Usar todo.py list <listname>

Removendo itens acabados de uma certa lista
Use todo.py donein <listname> <id> Para obter o ID, usetodo.py list <listname>

Criação de uma nova lista
Usar todo.py create <listname>

Removendo uma lista
Usar todo.py delete_list <listname>
